# 内容提炼与文风复刻：验收评分表（Rubric, 0-5）

用途：
- 用同一套标准验收：领域提炼、理念/偏好、心智模型、文风 DNA、文风复刻文本
- 支持迭代：每次复测都能定位偏差项与修订策略

评分规则（通用）：
- 0 分：缺失/明显错误/不可用
- 1 分：有但很弱，依赖大量主观补完
- 2 分：基本可用，但证据或边界明显不足
- 3 分：可用且一致，少量问题不影响使用
- 4 分：高质量，结构清晰，能稳定复现
- 5 分：接近“可产品化”，对新情境也稳健

---

## 1. 总体验收门槛（建议）

必须项（任一不满足则不通过）：
- 禁忌词违例 = 0（文风复刻文本）
- 每条关键结论可追溯到证据（领域/理念/模型/文风都需要）
- 至少 3 个心智模型通过三重验证中的 2 项（跨主题复现/生成力/排他性）

推荐门槛：
- 总分（加权）≥ 24/30
- 文风盲测：至少 60% 评审者判断“复刻文本更像该账号”或“难以区分”

---

## 2. 评分维度（0-5）与扣分点

建议 6 个维度，每项 0-5 分，总分 30 分。你也可以按项目权重调整。

### 2.1 可追溯性（Evidence Traceability）

衡量：
- 任意结论是否能回链到证据段落/claim
- 证据是否足够覆盖结论（不是“贴一句原文就算证明”）

扣分点：
- 结论没有证据
- 证据与结论关系弱（原文没这个意思）
- 证据集中在单一文章，缺乏跨内容支撑

### 2.2 领域结构（Domain Map Quality）

衡量：
- taxonomy 是否可导航（读者能从典型问题找到答案路径）
- 主题命名是否具体、可操作（避免空泛词）
- 概念口径是否统一且可复用

扣分点：
- 主题层级混乱或重复
- 大量“方法论/思考/提升”这类空泛标签
- 概念定义像百科，不像账号口径

### 2.3 观点谱系（Beliefs & Preferences）

衡量：
- 是否提炼出稳定的核心信念、反对清单、受众假设
- 偏好矩阵是否包含阈值条件/例外（不是一刀切）

扣分点：
- 只罗列观点，没有“取舍机制”
- 红线与反对项不清晰
- 缺少边界与反例

### 2.4 模型可运行性（Model Operability）

衡量：
- 模型是否可用于推断新问题（有推理链）
- 变量关系与触发条件是否明确
- 有边界与失效条件，避免泛化

扣分点：
- 模型是口号/鸡汤，无法落到变量与规则
- 触发条件含糊（“任何时候都适用”）
- 无反例或把反例硬解释掉

### 2.5 模型验证质量（三重验证）

衡量：
- 跨主题复现：至少 2 个子议题出现同一关系或规则
- 生成力：对验证集新问题能推断且自洽
- 排他性：与行业共识/泛建议可区分

扣分点：
- 只有单篇证据支撑
- 推断结果更像通用建议
- 排他性来自“用词怪异”，而非视角独特

### 2.6 文风一致性（Style Consistency）

此项由“量化 + 人工”两部分共同决定，建议各占 50%。

量化部分衡量：
- 关键指标命中率（见第 3 节）
- 结构模板命中（是否按账号常用骨架组织）
- 关键词覆盖与概念口径一致
- 禁忌词违例=0

人工部分衡量：
- 盲测判断“像不像”
- 语气、节奏、用词是否自然（不过度模仿）

扣分点：
- 语气整体偏离（过软/过硬/过鸡汤）
- 结构不符合账号习惯（比如账号结论先行，你却长铺垫）
- 概念口径跑偏（同词异义）

## 3. 文风量化阈值：如何设“基线区间”

步骤：
1) 抽样：随机抽取不少于 20 段原文段落（尽量覆盖不同主题与时间）
2) 统计：对每段计算指标，汇总均值与分位数（优先 P25-P75）
3) 分级：把指标分成“必须命中”和“可容忍偏离”

建议指标集（最小可用）：
- 平均句长（字/句）
- 短句比例（<=15字句子占比）
- 疑问句比例
- 转折频率（但是/不过/然而/但 的出现频率）
- 因果连接频率（因为/所以/因此）
- 第一人称比例（我/我们）
- 第二人称比例（你/你们）
- 确定性语气比例（一定/显然/必须） vs 谨慎语气比例（可能/也许/大概）
- 类比/例子密度（每千字）

必须命中（建议）：
- 平均句长落在 P25-P75
- 结构模板命中（选定模板后不得偏离骨架）
- 禁忌词违例 = 0
- 关键词/概念口径一致（同词同义）

可容忍偏离（建议）：
- 类比/例子密度（允许按主题变化）
- 第一/第二人称比例（允许按文体变化：短帖/长文/脚本）
- 确定性/谨慎语气比例（允许微调，但不得反转账号气质）

---

## 4. 人工盲测：组织方式与通过标准

推荐方式：
- 每次盲测使用同主题的 A/B 两段（原文 vs 复刻）
- 至少 3 位评审者（你自己也可以，但最好有外部读者）

评分维度（每项 0-5）：
- 整体像不像
- 结构像不像
- 语气像不像
- 用词像不像（概念口径）
- 节奏像不像（段落长度与推进方式）

通过标准（建议）：
- “整体像不像”平均 ≥ 3.5
- 或者：至少 60% 评审者认为“复刻更像/难以区分”

---

## 5. 最终验收清单（可直接勾选）

内容提炼：
- 领域地图：2-3层 taxonomy + 每个叶子主题 3 个典型问题
- 概念口径：高频概念的账号定义 + 边界 + 误用纠偏
- 观点谱系：核心信念 + 反对清单 + 偏好矩阵 + 受众假设
- 心智模型：3-7 个模型卡片（含证据、边界、反例、三重验证记录）

文风与验证：
- 文风 DNA：句式指纹指标表 + 结构模板库 + 关键词/禁忌词/口癖
- 文风复刻文本：同事实改写 1 篇 + 新问题写作 1 篇
- 量化验证表：关键指标命中结果 + 偏差原因 + 修订建议
- 盲测结果：评审打分表 + 结论（是否通过）+ 下一轮修订点
