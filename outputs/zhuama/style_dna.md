### C类模式识别

#### 常见标题结构

- `场景 / 对象 + 议题`：先给一个可视化场景或人物，再补一个价值议题或成长命题。
- `问题句 + 关系冲突`：用“什么 / 为什么”把读者带进一个关系或控制问题。
- `作品名 + 当代解释`：先点作品，再补“控制 / 选择 / 人性 / 自我守护”等当代翻译。

#### 常见标题抓手

- 场景抓手优先：先让读者看见一个戏剧情境、成长处境或行动画面。
- 议题抓手次之：再把场景翻译成“控制 / 选择 / 真实 / 牺牲 / 成长”等判断命题。
- 轻冲突而非猛冲突：会用“劫持 / 冲突 / 控制 / 迷失”制造张力，但通常不靠极端情绪词。

#### 常见冲突模板

- `你以为的关系，其实是另一种控制`
- `看似温柔 / 合理 / 熟悉的事物，背后藏着危险结构`
- `古典故事 / 戏剧情境，其实正对应当下困境`

### D类禁忌指标

- 禁用空泛宏大词作为主抓手：如`时代洪流`、`颠覆未来`、`重塑认知`、`赋能成长`。这些词没有具体场景，和该账号的“对象/场景先行”口径冲突。
- 禁用廉价焦虑式标题：如`再不关注孩子就晚了`、`90%家长都做错了`、`不看一定后悔`。该账号会制造张力，但不会靠恐吓抢点击。
- 禁用与正文判断无关的纯噱头标题：如正文讲“主体性与控制”，标题却写成`结局太炸裂了`。该号标题必须和正文议题同口径。
- 禁用纯方法清单型标题：如`3个方法解决亲子问题`、`5步提升孩子表达力`。这会把账号拉向工具号，而非议题型写作。

### C类模式识别

#### 常见开头方式

- `场景压入式`：先把读者放进一个具象画面，再让议题自然显形。
- `故事托底式`：先给人物、动作、关系，再逐步抬出判断。
- `轻判断补位式`：不会一上来就高压下定义，但会在第一屏内补一个清晰立场。

#### 常见首段动作

- 给场景：如荒原、洪水、山洞、舞台、家庭关系等具体空间。
- 给人物处境：先让孩子、家长、角色处在一个紧张关系里。
- 给问题而不是给答案：用提问和情境把读者卷入，而不是直接科普。
- 叙事后立题：先带入，再用一句判断把读者钉在议题上。

### D类禁忌指标

- 禁用宏观空铺垫：如`随着时代发展，教育越来越重要`、`在信息时代，我们都面临挑战`。这类开头没有人物和场景，明显不像该账号。
- 禁用长背景说明后才进入主题：连续两三段只讲项目背景、课程安排、作品历史，而不让读者进入冲突或关系。
- 禁用与标题承诺不一致的绕路开头：标题在讲控制、成长、牺牲，开头却先花大段篇幅介绍活动时间、机构信息或作者感想。
- 禁用开头即下结论式说教：如`家长一定要明白……`、`我们必须意识到……`。该号会给判断，但通常先托场景、人物或问题。

### 典型样本

- [小毛头](file:///Users/hymanhai/Documents/zrhp/content-distil/docs/zhuama/当少年踏上百年逃亡路%20——《小毛头》里的生存、选择与人性微光.md) 开头：`想象一个场景：` / `前路是未知的危险，身后是被洪水吞噬的家园，他们要靠一双脚，走完近两个月、四百多公里的逃亡路。` 典型性在于先把读者扔进场景，再让议题自己压上来。

### 生成约束

- 开头要尽快进入核心判断或核心冲突，不允许长时间停留在背景铺垫。
- 开头作为高感知层，需保证“第一眼像”，但不得用夸张语气代替真实风格。

### C类模式识别

同账号在该目录里存在多种文体，但目前可归为 3 个主模板：

#### 模板 A：议题拆解型（01/02/03 分节）

常见于：作品介绍 + 议题深挖（如《牺牲》《旅途》的阐释文）。

骨架：
- 引子：用情境或意象把人带入冲突
- 立主题：用一句“最核心/最根本/本质”把议题钉住
- 分节拆解：`01 / 02 / 03` 每节一个子议题
- 每节结构：概念或框架 → 当代映射 → 对读者的反问或追问
- 结尾：回到“成长/自我/责任/主体性”，再落一个行动召唤

#### 模板 B：强隐喻推进型（意象贯穿）

常见于：《旅途》相关文本、心理/关系议题文本。

骨架：
- 抛出意象：影子、风、网、钉子、吞噬
- 解释意象背后的心理结构：依恋、控制、规训、占有
- 把意象投射到当下：算法、社群、成功学、家庭期待
- 用 2-3 个读者自检问题收束：你在乎什么、你为谁献祭、你交出了什么

#### 模板 C：招募/活动型（先愿景后信息）

常见于：招募、活动、课程、工作坊。

骨架：
- 先抛愿景：戏剧不只是表演，而是理解世界、认识自己
- 给孩子或家长的价值：敏感不是弱点，需要一个空间，练习看见
- 再给活动安排、报名方式等信息块

### D类禁忌指标

- 禁用结构松散、只堆观点不推进
- 禁用纯信息罗列而缺少议题化表达
- 禁用鸡汤式口号收尾
- 禁用硬转化压倒正文逻辑
- 禁用“万能总分总”模板硬套：开头抛结论，中间机械列三点，结尾再喊一次口号，会抹掉该账号的戏剧推进感。
- 禁用信息块提前吞没正文：尤其是招募/活动文，不允许报名信息在前、价值和议题在后。
- 禁用结尾只剩报名/购票 CTA：该号允许转化，但不能让结尾失去判断、陪伴或邀请探索的收束动作。

判定提示：
- 情绪强度：1 分为近乎无情绪温度，3 分为有稳定情绪色彩，5 分为情绪牵引非常明显。
- 吐槽浓度：1 分为完全不吐槽，3 分为轻微点破空话，5 分为吐槽成为显著语气动作。
- 焦虑制造程度：1 分为不制造恐慌，3 分为局部有警醒，5 分为大量放大风险和压力。
- 安抚程度：1 分为几乎不托住读者，3 分为有基本安抚，5 分为明显提供安全感和容纳感。
- 权威感：1 分为情绪先于判断，3 分为语气稳，5 分为有明确判断支撑的权威感。
- 陪伴感：1 分为距离很远，3 分为有同行感，5 分为像有人在旁边陪你拆问题。
- 战斗感：1 分为完全不反驳，3 分为会温和反驳，5 分为持续对行业空话或错误做法开火。

### C类模式识别

#### 常见情绪动作

- 先托住再推进：先给安全感、陪伴感，再把读者带进更难的议题。
- 先点破再解释：会直接指出控制、规训、裹挟，但通常紧接着解释其结构。
- 先共情再邀请行动：不会让情绪停在痛感上，而会把它带向理解、探索或进入剧场。

#### 常见否定方式

- 否定现象，不羞辱个人：批评的是控制逻辑、社会结构和空话，不是把读者钉成坏人。
- 否定靠重定义完成：不直接骂，而是把“爱 / 正义 / 成熟”重新翻译成另一种结构。
- 否定后给容纳空间：指出问题之后，仍保留“可以害怕 / 可以犹豫 / 可以重新理解”的空间。

#### 常见收束情绪

- 收束到陪伴：如“让我们一起”“一起探索成长的无限可能”。
- 收束到判断：把情绪落回主体性、边界、选择、自我守护。
- 收束到邀请：常把文章末尾处理成进入剧场、继续探索、继续思考的同行式邀请。

### D类禁忌指标

- 禁用羞辱读者：如`你还不明白就是认知太低`、`家长都是问题根源`。该号可以批评控制，但不攻击具体人。
- 禁用廉价恐吓：如`再不改变孩子就废了`、`你正在毁掉孩子一生`。这和该账号的低焦虑边界冲突。
- 禁用只输出怒气、不输出解释和方案：如果点破规训、控制、空话，后面必须给出理解框架或行动方向。
- 禁用过度煽情：不靠连续感叹、惨烈形容词和情绪堆叠制造“感人”，而是靠案例与结构形成情绪。

### 典型样本

- [新班招募](file:///Users/hymanhai/Documents/zrhp/content-distil/docs/zhuama/抓马教育新班招募丨牵起小手，来戏剧故事里做场冒险游戏.md)：`这里很安全。` / `害怕是可以的，想念是正常的。` 典型性在于情绪先托住人，再继续推进成长议题。
- [旅途深度探索](file:///Users/hymanhai/Documents/zrhp/content-distil/docs/zhuama/七年回望，戏剧照见成长｜《旅途》：一场关于依恋、控制与自我守护的深度探索.md)：`很多母亲根本不会承认……这部剧把这层纸捅破了。` 典型性在于允许直接点破控制，但仍然在批评现象，不羞辱具体人。
- [《牺牲》开票](file:///Users/hymanhai/Documents/zrhp/content-distil/docs/zhuama/《牺牲》开票丨神从未现身，但每个人都在替他说话.md)：`让我们一起走进剧场……共同探索自我与家庭、社会的关系。` 典型性在于结尾常把判断收束为同行式邀请，而不是高压号召。

### 生成约束

- 允许直接否定错误做法，但不攻击具体人。
- 允许吐槽空话，但最终必须落回解释、判断或解决路径。

## 9. 权重说明（用于新版 Rubric）

文风 DNA 八层评分采用不等权重，总分 100：

- 观点 DNA：25%
- 结构 DNA：20%
- 开头 DNA：15%
- 案例 DNA：15%
- 词汇 DNA：8%
- 句法 DNA：7%
- 情绪 DNA：6%
- 标题 DNA：4%

权重解释：
- `观点` 决定文章有没有这个号的判断力。
- `结构` 决定文章展开方式像不像。
- `开头` 决定读者第一眼是否觉得像。
- `案例` 决定文章是否有真实感和账号经验感。
- `词汇 / 句法 / 情绪 / 标题` 是重要辅助层，但不应主导整体文风。

## 10. 当前最小验收要求

后续任何“新写作/改写”建议至少做以下自检：

- 句长：平均句长落在 [25, 63]
- 结构：明确选用模板 A/B/C，并且骨架不跑偏
- 概念口径：同词同义，例如“神谕 / 大局 / 结构性暴力 / 自我规训”的用法一致
- 观点：至少完成一次问题重定义，并给出判断标准或行动路径
- 案例：至少出现一个具体角色、具体场景、具体冲突
- 情绪：避免纯鸡汤式鼓励，只给情绪、不提供结构与冲突
