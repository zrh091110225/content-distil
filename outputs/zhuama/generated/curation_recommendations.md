## 2. validation_report 复测建议

目标：
- 在不重复现有 A/B 样本的前提下，优先补足模型的“跨主题复现”和“边界/反例”。

### 2.1 建议优先复测的 3 篇

#### 优先级 1

- [用七年探索，这个时代最迫切的命题——真实与想象的冲突](file:///Users/hymanhai/Documents/zrhp/content-distil/docs/zhuama/用七年探索，这个时代最迫切的命题——真实与想象的冲突.md)
- 建议复测模型：`M1 神谕识别模型` + `M4 象征距离模型`
- 推荐理由：
  - 当前 `validation_report.md` 还没有覆盖“其他”主题。
  - 这篇能把“真实 / 想象”“经典结构 / 当代意义危机”拉到一起，是 M4 最值得补强的一篇。
  - 同时又能把 M1 从“家庭 / 牺牲”扩展到“意义危机 / 信息现实 / 外部期待”的更宏观层。

#### 优先级 2

- [观后感：希望的光芒冲破《牺牲》](file:///Users/hymanhai/Documents/zrhp/content-distil/docs/zhuama/观后感：希望的光芒冲破《牺牲》.md)
- 建议复测模型：`M1 神谕识别模型`
- 推荐理由：
  - 当前验证集里的“牺牲”主题只测了议题拆解型正文，还没测“观后感 / 受众接受型”文本。
  - 这篇很适合检验：当文本不再是作者论证，而是观演后的再阐释时，M1 还能不能稳定解释。
  - 如果能解释，说明模型不依赖单一文体；如果解释力明显变弱，也能帮助修正模型边界。

#### 优先级 3

- [荒土的风，吹醒“孤独”的成人——为什么我们会被《旅途》打动？](file:///Users/hymanhai/Documents/zrhp/content-distil/docs/zhuama/荒土的风，吹醒“孤独”的成人——为什么我们会被《旅途》打动？.md)
- 建议复测模型：`M4 象征距离模型` + `M2 温柔陷阱模型`
- 推荐理由：
  - 当前旅途主题已验证 M2/M3，但“成人为什么会被童话/荒土叙事打动”这一层还没有单独展开。
  - 这篇更适合验证 M4 在“成人共鸣”和“共享世界中的行动形式”上的解释力。
  - 若 M2 也能稳定命中，说明“依恋 / 控制 / 温柔陷阱”不仅属于儿童语境，也能外溢到成人经验。

## 3. 建议执行顺序

### 3.1 主文档维护

- 第一步：把“标题 / 词汇 / 案例 / 情绪”4 层优先增补写进 `style_dna.md` 的典型样本区。
- 第二步：如你希望保持每层只留 3 条样本，则采用“替换”而不是“追加”。
- 第三步：结构层本轮不动，避免样本越写越多、信息开始失焦。

### 3.2 验证报告维护

- 第一步：在 `validation_report.md` 新增 v2 小节。
- 第二步：按 `其他 -> 牺牲观后感 -> 旅途成人共鸣` 的顺序补 3 篇复测。
- 第三步：每篇复测都记录：
  - 核心结构
  - 模型命中情况
  - 哪些地方是补强证据
  - 哪些地方构成模型边界或反例
