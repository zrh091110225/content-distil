# 七年回望，戏剧照见成长｜《旅途》：一场关于依恋、控制与自我守护的深度探索

> 公众号: 抓马教育
> 发布时间: 2026-05-21 18:03:48
> 原文链接: https://mp.weixin.qq.com/s/MX6so7AjcWk5WJ4F-iHNcQ

---


2019 年，由英国著名剧作家Chris Cooper创作，抓马教育倾力打造的《旅途》（The Journey）在菊隐剧场正式上演。《旅途》以虚构帝国为背景，讲述被遗弃的北方儿童在荒漠中的欺骗与控制下，踏上寻找自我与归途的艰难旅途。区别于传统儿童剧，它不回避沉重与真实，而是让孩子在沉浸式体验中直面恐惧、依恋与挣扎，在故事里理解 “被吸引”“被包裹”“被吞噬” 的深层感受。

七年后，回望这段创作历程，这部直面权力、依恋、独立与控制的作品，依旧以厚重而锐利的力量，影响着每一位走进戏剧的孩子与教育者。我们采访了七年前《旅途》的导演Coco，她同时也是当时“母亲”角色扮演者，为我们讲述她与《旅途》的故事，同时也从她的角度出发，探索：为什么在2026年的当下，我们要重排这部剧？

![Image](images/img_001.jpeg)

《旅途》2019

**Q1**

**当年排演这部剧时，**

**你最大的压力是什么？**

最让我焦灼的，其实是孩子们年龄比较小。他们在故事探索时，完全是沉浸式的过程戏剧；但把它转化成 “演出”，其实是两种完全不同的语言，对我来说也是一个很大的挑战。

**Q2**

**七年前你理解的 “妈妈”，**

**和现在有什么不同？**

七年前，那时候我还没有成为母亲，我只是在 “扮演” 母亲。真正的妈妈是慢慢长出来的，不是怀孕了、别人叫你妈妈，你就是妈妈。我不愿意把它简单说成“母爱伟大”，它的确伟大。如赵婷导演提到的理想中的母亲，如《哈姆奈特》的艾格尼丝，是大地之母式的存在：无条件看见孩子、支撑灵魂、提供安全感，而非温柔表象下的占有与吞噬。母亲身份是 “慢慢长成” 的，不是天生完美，而是在陪伴、直觉、觉察中一点点生长出来的**。**

![Image](images/img_002.jpeg)

《旅途》2019

《旅途》也在警惕：任何你熟悉的东西，都可能以 “母亲” 之名实施控制，人要守住自我，不被吞噬、不交出灵魂与思想。而当年这部剧，其实一直在提醒我们、让我们学习，甚至保持警惕——在育儿里，你是不是也在使用某种 “控制”？我们是不是也在用幻象跟孩子相处？像 “听妈妈的话” 这类表达，大家都觉得是好的、是爱。但往深一层想：到底是孩子需要，还是你更想成为 “完美母亲”？

**Q3**

**剧中的 “母亲”**

**到底是一个怎样的角色？**

我认为编剧Chris写得非常犀利、一针见血。很多时候我们以为是 “关心”：替孩子决定穿什么、吃什么、走什么路。你以为是爱，可从另一个角度看，这就是控制，是一种暴力。最大的暴力，就是替别人决定思想。

家长常常困惑：孩子为什么不跟我说话了？为什么沉默了？是什么让他闭上嘴？

![Image](images/img_003.jpeg)

《旅途》2026

这部剧厉害的地方就在这里：无论你多大年纪，都能在 “母亲” 这个角色里看见自己。剧里有很多细节特别真实：“妈妈”不断喂饭、用勺子分食物，孩子们排着队；她永远温柔、不严厉，长得漂亮，像知识分子，扮演一个完美理想型。所有人都会迷恋她。

她出场很华丽，有学识、有光环，不是严厉的母亲，而是动人且极具吸引力。孩子会不由自主地依恋她。而她，也在利用这份依恋。在开场第一场戏，就是一场赤裸裸的交易—— 她在和男人倒卖孩子，在数钱、掂量重量、算价格。一个受过良好教育、能把 “母亲” 演得这么像的人，灵魂去哪了？一个有趣的部分是，这个邪恶的女人在扮演母亲中的时候，她应该是全情投入的，她甚至可能自己都没意识到，她只是在扮演母亲。就像一个人贩子在跟拐卖的孩子相处中，慢慢地相信自己是真的妈妈了。我真的相信有很多男人也会羡慕做妈妈的体验，这太有意思了。

而这对孩子也特别重要：人生路上会遇到很多你 “忍不住想拥抱” 的东西，它不是诱惑，但你必须学会识别。

**Q4**

**你印象比较深刻的是？**

我印象最深的，是对 “母亲” 这个角色的探索。那是 2019 年，我还没当妈妈，却要去扮演一个“母亲”。Chris跟我说：你要好好思考母亲的身份。 其实在问我：**你准备好做妈妈了吗？**

你离不开对母亲的爱与依恋，但你必须思考：依恋背后，爱背后，到底藏着怎样的控制与被控制？

![Image](images/img_004.jpeg)

《旅途》2019

演出时，当时星洲（现为青年剧团成员）饰演的代北，把头深深埋进我怀里。当时我扮演的“母亲”，袖子设计成水袖—— 它能容纳万物，也像一张嘴，把孩子“吞”进去、抱住、控制、占有。这个设计非常精妙。

**Q5**

**这部剧对你现在的生活和养育，**

**有真实影响吗？**

对我最大的影响，就是对思想控制、思想暴力这部分的觉察**。**

它真的帮到我现在做妈妈。我会不断问自己：我是真的为孩子好，还是我觉得 “这个东西很好”？

太难分辨了。比如给孩子穿衣服，他明明不想穿，我还是会强行把胳膊塞进去。那一刻，教养已经开始了 ——强迫与压力已经存在。很多母亲根本不会承认，甚至自己都不知道自己在控制。这部剧把这层纸捅破了。

**Q6**

**放在2026 年的今天，**

**这部剧为什么仍然值得推荐？**

![Image](images/img_005.jpeg)

《旅途》2019

先从海报说起。2019 年我们那版，视觉符号是妈妈的眼睛。那双眼睛很美，却能摄人心魄，把别人的灵魂吸走；妈妈的眼睛里有路，是你被引导、被选择走的路。

![Image](images/img_006.jpeg)

《旅途》2026

我看到2026新版海报，第一眼就认出：那应该是妈妈的头脑。我发现，不变的部分依然是 —— 警惕“什么在占有你的头脑、占有你的精神与灵魂”。

回到剧本身，《旅途》和2026 当下的连接更强了。AI 来了，信息爆炸，连刚出生的婴儿都会对屏幕执着，人太容易被各种东西缠绕、占据。我们更需要问：你怎么跟自己保持连接？你知不知道自己在想什么、在乎什么？

**风一吹就被带走，那不叫影响，那叫被掠夺。**《旅途》对于当下依然极具现实意义：它讲勇气，更讲你要像钉子一样把自己 “钉住”。现在太多东西披着 “母亲” 的壳：情绪煽动、情感渲染、深层依恋…… 都会让人失去本来、真实的样子。你要跟自己做一个决定：是随风飘走，还是像晨曦一样钉住自己——跟自己立约，跟自己达成默契。不让时代的洪流、内在情绪的微波把人淹没。

**Q7**

**这部剧最值得家长和孩子**

**一起看的点是什么？**

关于母亲的部分，以及和当下这个时代的连接。

它适合每一个年龄层的人重新看待 “母亲” 这件事。

它表面没在讲母亲，却处处都在讲母亲；它讲的不是亲情，而是思想、占有、控制与自我。这部分最深刻，也最有影响力。
